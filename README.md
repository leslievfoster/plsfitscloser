# plsfitscloser
MATLAB code accompanying a paper comparing partial least squares (PLS) with principal component regression (PCR)

# AUTHOR
Leslie Foster, Department of Mathematics, San Jose State University

# REFERENCE:
"In the 2-Norm, PLS Fits Closer Than or the Same As PCR"

# SOFTWARE LANGUARE

MATLAB 2024b

# REQUIREMENTS

This code requires the following package:

(1) IRTools by James Nagy, Silvia Gazzola, and Per Christian Hansen
             https://github.com/jnagy1/IRtools

(2) The MATLAB symbol manipulation toolbox (optional)
