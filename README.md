#plsfitscloser
MATLAB code accompanying a paper comparing partial least squares (PLS) and principal component regression (PCR)

#AUTHOR:

Leslie Foster, Department of Mathematics, San Jose State University

#REFERENCE:
"In the 2-Norm, PLS Fits Closer Than or the Same As PCR"

#SOFTWARE LANGUAGE:

MATLAB 2024b

#REQUIREMENTS:

This code requires the following package or toolbox:

(1) IRTools by James Nagy, Silvia Gazzola, and Per Christian Hansen
         https://github.com/jnagy1/IRtools, for generate_figure1_figure2

(2) The MATLAB symbol manipulation toolbox, for check_paper_equations
